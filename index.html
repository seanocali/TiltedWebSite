Game Manager & Frontend

Available On:

Features

Emulation

Integrated support for RetroArch, RPCS3, ScummVM and more. Easy to add other emulators, or use RocketLauncher.

Steam

Maintenance-free, always up-to-date library sync for both Steam and non-Steam PC games. Supports categories.

Skins

Use built-in graphical tools for quick and easy designs, or go for full-blown advanced customization with WinUI XAML.

Playlists

Use auto-generated playlists and/or custom ones to present your game library just the way you want. And yes, they support nesting!

Smart Library

Set up your library with just a few clicks. Games are detected automatically. Proper file naming and organization is optional!

Blazing Fast

Even with thousands of games, the initial game library build only takes a few minutes. Tilted UI startup only takes a few seconds.

Quicklaunch

A Quicklaunch overlay with dynamic playlists based on favorites, ratings, and play history.

Search

Advanced search with instant results for opening a specific game or playlist without the need to navigate.

Input

UI specifically designed to support navigation using analog sticks, touchscreen gestures, arcade trackballs and spinners.

Tilted Database

A massive offline game database is included with Game Manager, but you can also provide your own if you wish.

Critic Ratings

Metacritic scores are automatically pulled for PC games, and the Tilted Database includes thousands of scores for retro games.

important_devices Second Screen

Run Tilted UI on additional devices as a remote control and have handy access to game instructions.

HyperSpin Themes

Tilted Media Tool lets you import HyperSpin themes, converting them into HTML5 files for use in Tilted UI.

Cross-platform

Run Tilted Game Manager and Tilted UI on your Windows, macOS, or Linux gaming system. Tilted UI also works on Android and iOS.

api Web API

Tilted Game Manager can give browsers and 3rd party software access to everything. Turn a cheap tablet into a digital marquee!
