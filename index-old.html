Features

Smart Library
Build your entire game library in just a few clicks. Instead of setting up one platform at a time, your game files are automagically identified at blazing fast speed.

Custom Skins

Creating skins for Tilted UI is as easy or complex as you want it to be. In-app customization controls can be saved out as a configuration file. Use this file by itself as a new skin, or as the starting point for something more. Every aspect of the UI can be altered or even redsigned from scratch using XAML markup files.

Multiplatform

Tilted UI & Tilted Game Manager are available for Windows, Linux, and macOS.
Tilted UI is also available for Android and iOS.
